# Portail Régional Sud-Est

Maquette React / TypeScript / Vite d'un portail interne régional SharePoint / M365 pour la Direction Régionale Sud-Est.

## Objectif

Le portail sert de point d'entrée régional pour retrouver rapidement :

- les outils clés ;
- les opérations commerciales ;
- les supports et documents utiles ;
- les fournisseurs et fabricants ;
- les contacts et expertises ;
- les remontées terrain ;
- l'onboarding et les procédures.

Il ne remplace pas Pixel, Qlik, CRM, Soon, Yoobic ou les autres outils métier. Il sert à orienter, prioriser et rendre l'information plus lisible.

## Lancer le projet

```bash
npm install
npm run dev
```

Build :

```bash
npm run build
```

Tests :

```bash
npm test
```
